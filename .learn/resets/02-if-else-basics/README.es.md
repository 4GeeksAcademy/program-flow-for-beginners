# If Else Basics

Escenario: Estas en una fiesta. Una persona solo puede beber alcohol si tiene 18 o mas. Crea el flujo de decision para validar edad, ruta permitida, ruta no permitida y fin.

Crea un diagrama de flujo en Mermaid en `answer.js` que cumpla la logica requerida para este ejercicio.

Usa identificadores de nodos como A, B, C y conectalos con flechas.
