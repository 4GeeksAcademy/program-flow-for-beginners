# Bienvenida: Como Funciona Este LearnPack

En este proyecto, cada ejercicio te pide resolver un escenario logico enviando un diagrama Mermaid.

## Donde escribir tu solucion

Edita solo `app.js` dentro de cada carpeta de ejercicio.

## Flujo recomendado

1. Usa esta herramienta desarrollada por un estudiante graduado de 4Geeks: https://waficmikati.github.io/mermaid/
2. Genera el diagrama alli.
3. Copia y pega el codigo Mermaid en `app.js`.

## :bulb: Pista
- No necesitas dominar la sintaxis de Mermaid en este tutorial.
- Enfocate en la logica que pide cada ejercicio.
- Si fallan los tests, compara tu resultado con `rubric.json`.
