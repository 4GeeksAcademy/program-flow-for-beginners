# `01` Sequence Basics

Escenario: Preparas cafe antes de clases. Define un flujo lineal: inicio, hervir agua, preparar cafe, servir, fin. Sin decisiones.

## :memo: Instrucciones
1. Abre `app.js` en esta carpeta del ejercicio.
2. Escribe o pega tu flujo Mermaid dentro del template string.
3. Manten los ids de nodos (`A`, `B`, `C`, ...) y las conexiones segun la rubrica.
4. No renombres `answer` ni `module.exports` en `app.js`.

## :bulb: Pista
- Tu grafo debe incluir estos conceptos clave: inicio (start), fin (end).
- Busca tener al menos 3 conexiones antes de correr tests.
- Este ejercicio no requiere etiquetas de condicion en ramas.
- Evita patrones prohibidos en el texto del diagrama: while, for.
