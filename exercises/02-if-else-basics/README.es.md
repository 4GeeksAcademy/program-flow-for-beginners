# `02` If Else Basics

Escenario: Estas en una fiesta. Una persona solo puede beber alcohol si tiene 18 o mas. Crea el flujo de decision para validar edad, ruta permitida, ruta no permitida y fin.

## :memo: Instrucciones
1. Abre `app.js` en esta carpeta del ejercicio.
2. Escribe o pega tu flujo Mermaid dentro del template string.
3. Manten los ids de nodos (`A`, `B`, `C`, ...) y las conexiones segun la rubrica.
4. No renombres `answer` ni `module.exports` en `app.js`.

## :bulb: Pista
- Tu grafo debe incluir estos conceptos clave: inicio (start), entrada (input), valido (valid), invalido (invalid), fin (end).
- Busca tener al menos 3 conexiones antes de correr tests.
- Usa etiquetas de rama exactamente como se esperan (acepta sinonimos): si (yes), no.
- Manten nombres y etiquetas simples para que coincidan con el validador.
