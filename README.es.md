Ejercicios interactivos y auto-evaluados para practicar logica de programacion con Mermaid.

## Instalacion en un clic (recomendado)

Abre en [Codespaces](https://codespaces.new/) y ejecuta:

```bash
learnpack start
```

## Instalacion local

1. Instala LearnPack:
```bash
npm i @learnpack/learnpack -g
```
2. En la carpeta donde esta `learn.json`, ejecuta:
```bash
learnpack start
```

## Estructura de ejercicios

Cada carpeta de ejercicio incluye:

1. `app.js`: Archivo principal editable con Mermaid.
2. `README.md`: Instrucciones en ingles.
3. `README.es.md`: Instrucciones en espanol.
4. `rubric.json`: Reglas de validacion usadas por los tests.
5. `test.js`: Entrada de tests del ejercicio.

`00-welcome` solo contiene readme y no tiene archivos de test.

## Notas

- Usa la extension de LearnPack para VSCode para mejor auto-apertura.
- Los tests son estrictos; usalos como guia.
